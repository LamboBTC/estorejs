#EStoreJS 0.2.10

EStoreJS is a framework written in [Node.js](http://nodejs.org) for running
eCommerce stores.

Highlights:

* Runs on MongoDB
* Uses the [KeystoneJS](http://keystonejs.com) framework.
* Designed to work with other applications and services.
