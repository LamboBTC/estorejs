EStoreJS 0.2.10

EStoreJS is a framework written in Node.js for running eCommerce stores.

Download the latest version here.

Highlights:

  • Runs on MongoDB
  • Uses the KeystoneJS framework.
  • Designed to work with other applications and services.

Quickstart

You will need a PaaS host that supports Node.js apps and a MongoDB server. EStoreJS is deployed entirely through git, the hosting provider should have instructions on how to do this.

Known PaaS services/tools:

  • Nodejitsu

  • Dokku (Run your own mini Heroku)
